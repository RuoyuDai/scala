1. [基础语言] 如下Scala代码，

```scala
val x = List(1, 2, 3, 4, 5) match {
  case x :: 2 :: 4 ::  _ => x
  case Nil => 42
  case x :: y :: 3 :: 4 :: _ => x + y
  case h :: _ => h
  case _ => 101
}
```
x最终被赋值为：

>A.  1    
>B.  42     
>C.  3    
>D.  101
